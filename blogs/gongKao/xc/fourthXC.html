第四章、逻辑判断 ​

逻辑判断主要包括7类题型：翻译推理、组合排列、削弱、加强、日常结论、原因解释和真假推理

1. 翻译推理 ​

 题型特征： 题干或选项中出现多组“逻辑关联词”，如：“如果……那么……”、“只要……就……”、“只有……才……”、“所有”、“有些”、“……且……”、“……或……”等

解题思路

先翻译，后推理

  1. 将逻辑关联词按照翻译规则翻译为 ① → ② 的形式（①、②分别代表一句话）
  2. 按照推理规则进行推导

1. “必要条件”的翻译陷阱 ​

思维总结

例：在推进依法治国的进程中，大力宣传法制必不可少。
翻译：推进依法治国 → 大力宣传法制（“大力宣传法制”必不可少，放在箭头后）

2. “除非……否则……”的翻译 ​
案例分析【除非……否则……】

例：除非意志坚定，否则将中途放弃。
“除非……否则不……”的翻译为
例句中“否则”后没有出现“不”，可以在“否则”后加两个“不”，双重否定表示肯定，意思不变。
即：（除非）意志坚定，（否则不）不将中途放弃。
翻译：-中途放弃 → 意志坚定

思维总结

“除非……否则……”的翻译为：
“……，除非……” = “除非……否则……”

案例分析

例：航班延误4小时以上需要赔偿旅客，除非有不可抗力因素出现。
等价于：除非有不可抗力因素出现，否则航班延误4小时以上需要赔偿旅客。
翻译：-超过4小时赔偿 → 不可抗力出现

考点六、关于“有的” ​

逻辑内涵：“有的”表示至少一个，即 ≥ 1，包含3种可能性：

  1. 只有1个满足；2. 有一部分满足； 3.全部满足

同义关联词：有、有些、一些、某些、有一些……
翻译规则：
有的1是2，翻译：1 → 2
有的1不是2，翻译：1 → -2
特别提示： “有的”这个句式在翻译时，“有的”二字必须放在箭头前
翻译规则：
规则1：带“有的”翻译不适用逆否等价规则
规则2：有的1是2 = 有的2是1
例：有的马是动物 = 有的动物是马
规则3：有的1不是2 = 有的不是2的是1
例：有的动物不是马 = 有的不是马的是动物
规则4：
例：所有马都是动物 → 有的马是动物

思维总结

与不能相互推导

考点七、关于“并非” ​

逻辑内涵：
“并非”，即否定。当“并非”出现时，意味着否定后面所有的内容。

案例分析【并非】

例：并非明天会下雨
翻译为： -明天会下雨 = 明天不会下雨

同义关联词：
“并非”可以同义替换为：否、不、……是错的
推理规则：
“并非”一出现，后面全部“否”： “必然” 换 “可能”；“有的” 换 “所有”；“且” “或” 相互换。谓语动词最后否
注意：必然，也可以同义替换为“一定”；未必 = 不必然

考点八、推理结构相似 ​

例：在全球范围内，所有的大城市的生活成本都比较高，上海是一座大城市，因此上海的生活成本比较高
翻译：大城市 → 生活成本较高，上海 → 大城市，所以，上海 → 生活成本较高（即a → b, c → a, 所以 c → b）
推理结构相似的：要进入法院工作必须通过 国家司法考试，小王在法院工作，因此小王已经通过了国家司法考试
翻译：法院工作 → 通过国家司法考试，小王 → 法院工作，所以，小王 → 通过国家司法考试（即a → b，c → a，所以 c → b）
